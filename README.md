# Linux Environment Variables Export Command

This Bash script demonstrates how to define, export, and display environment variables in Linux using the 'export' command. It shows the difference between local and exported variables and how exported variables are inherited by child processes.

## Language

`bash`

## How to Run

Save the code as `env_vars_demo.sh`, make it executable (`chmod +x env_vars_demo.sh`), and run it (`./env_vars_demo.sh`).
